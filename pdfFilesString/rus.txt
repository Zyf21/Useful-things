Данный раздел посвящен языку программирования Java и всем сопутствующим
технологиям. Java на сегодняшний день самым популярным языком программирования,
который позволяет создавать различные приложения широкого спектра: веб-сайты и веб-
сервисы, десктопные программы, мобильные приложения для ОС Андроид, современные
приложения с богатым интерфейсом (Java FX). Java - универсальный кроссплатформенный
язык, поэтому приложения на Java будут работать на большинстве известных платформ
как Windows, Linux, MacOS.
